Random Quote Generator

Challenge Page

The hardest part for this, for me, was just finding a free quote API that was CORS-friendly. Spoiler: I couldn't. I wound up using a CORS proxy that, well, I'm not fond of but I definitely learned a lot about HTTP security while working on this. Also played around with CSS gradients, animations and masks to make the shifting, the wipe effect, and the loading indicator. Part of me wants to add a minimum delay between the click and actual load, but it seems to run slow enough because of the heroku passthrough as is. Oh, and I didn't actually use any CSS frameworks for this. Just FontAwesome for the twitter icon. Totally written by hand. Go me.
